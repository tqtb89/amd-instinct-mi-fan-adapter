## 使用说明

0. 打开blender，创建script
1. 粘贴 https://github.com/tqtb89/amd-instinct-mi-fan-adapter/blob/main/blender.script 的内容
2. 运行
3. 在场景中选中所有物体
4. 导出->stl->勾选selection only和apply modifiers
5. 使用3d打印机打印，开启支撑

拆掉顶部外壳后替换3d打印件，注意安装顺序：
1. 安装最细边的卡扣
2. 安装顶部卡扣
3. 安装最粗边的卡扣

（安装风扇时会被一个凸起的卡扣挡住，剪掉即可）
